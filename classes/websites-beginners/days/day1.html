Coding Websites with HTML/CSS Workshop Curriculum Day 1

Set up a Code Anywhere account.

  1. On the ‘Connection Wizard’ search for HTML and pick the ‘ubuntu 14.04’ option
  2. Name your project and click create
  3. After your project has been created scroll down on the page and click on the link that looks like this 'http://name-yourUsername.codeanyapp.com. (this is what allows you to see the project as you’re making it).
  4. Right click on the project name in the left-hand side
  5. Go down to create file and title it ‘index.html'
